# Projeto-Software
Este projeto implementa um Sistema de Gerenciamento de Tarefas, permitindo criar, listar e atualizar tarefas.
O objetivo é demonstrar práticas fundamentais de desenvolvimento, organização de código, versionamento e documentação, como solicitado no trabalho acadêmico.

O sistema inclui:

API básica para criação e gerenciamento de tarefas

Estrutura de projeto organizada

Testes automatizados

Pipeline simples com GitHub Actions

Diagrama de Caso de Uso

Documentação dentro da pasta docs/
